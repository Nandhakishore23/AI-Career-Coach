# AI-Enhanced Conversational Model for Interview Preparation

## Overview
A real-time AI-enhanced conversational model that simulates interviews, generates adaptive questions, and provides personalized feedback using NLU and deep learning.

## Tech Stack
- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express, MongoDB
- **AI Service:** Python, Flask, Gemini/HuggingFace

## Getting Started

### Prerequisites
- Node.js
- Python 3.8+
- MongoDB

### Installation

1. **Client**
   ```bash
   cd client
   npm install
   npm run dev
   ```

2. **Server**
   ```bash
   cd server
   npm install
   npm start
   ```

3. **AI Service**
   ```bash
   cd ai_service
   pip install -r requirements.txt
   python app.py
   ```
